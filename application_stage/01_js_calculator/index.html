Instructions: This is a simple calculator app that can take two operands and calculate their result based on which operator (or type of operation) was selected. Users can select from the following five (5) operators: Addition, Subtraction, Multiplication, Division and Square Root.

Note: To find the square root of a number, simply enter the number in the first operand input only.
